Add a Form with the following Inputs (and validators):

  1. Mail address (should not be empty and should be an email address
  2. A Dropdown which allows the user to select from three different subscriptions (Basic, Advanced, Pro)
  3. A Password field (should not be empty)
  4. A submit button

It should:

  1. Display a warning messagem if the form is invalid AND was touched.
  2. Display a warning message below each input if it's invalid.
  3. Upon submitting the form you should simply print the values of the form to the console.
  4. Optionally, display it in your template.
